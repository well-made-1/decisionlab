# Roadmap  
## Well Made Decisions Lab

---

## Phase 1 – Pilot (no-code)

- Multi-step structured agent (Voiceflow)
- Book + Blogs + Framework hybrid retrieval
- JSON output enforcement
- Soft gate + auto email (via Google)
- Basic governance documentation
- Initiate voice of the customer

---

## Phase 2 – Custom Web + Python Backend

Frontend:
- Next.js
- Structured report rendering
- Optional authentication (Supabase Auth)

Backend:
- FastAPI (Python)
- Supabase Postgres + pgvector
- Deterministic retrieval weighting
- Explicit citation validation

---

## Phase 3 – Observability & Evaluation

- Langfuse tracing
- Prompt version control
- Regression harness
- Latency & cost tracking
- Tie with Google Analytics to demand generation activity to understand marketability of paid product

---

## Phase 4 – Product Enhancements

- Aggregate, non-personalized prompt storage and summary analysis
- Persistent report storage (personalized, time constrained, tied to payment)
- PDF export (structured decision support document or board-ready presentation deck)
- Team collaboration mode
- Admin tuning dashboard
- Paid or gated offering (a virtual executive decision coach subscription or metered offering)

---

## Phase 5 – Advanced Intelligence

- Weighted retrieval tuning
- Framework clustering
- Pattern detection across decisions (with segmentation for different customer types)
- Confidence calibration scoring
